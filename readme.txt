With Press Tube you can easily access to YouTube content directly from your site administration panel.

== Description ==
Press Tube let you access YouTube content from your site administration panel, easily find them and embed in your posts content. You can access to your channel content and create custom playlists.

= Features =
* Easily select YouTube content from post edit page.
* Embed content in your posts with many options.
* Display YouTube playlists in various styles (default, list, slider, gallery).

= Shortcodes =
* [subscribe]: Display the YouTube subscribe button for a given Channel Id
* [playlist]: Display the playlist for a given Id that can be YouTube playlist Id or custom playlists Id
* [live-chat]: Display the live chat frame for a given Live video id.

= Widgets =
* YouTube LiveStream: Display your live stream video with chat and many options
* YouTube Videos: Display a set of videos that match your options in different styles
* YouTube Subscribe: Display the YouTube Subscribe button for your channel or any channel with many options

More to come in next versions.

== Installation ==

= From within WordPress =
1. Visit 'Plugins > Add New'
1. Search for 'Press Tube'
1. Activate Press Tube from your Plugins page.
1. Go to "after activation" below.

= Manually =
1. Upload the press-tube folder to the /wp-content/plugins/ directory
1. Activate the Press Tube plugin through the Plugins menu in WordPress
1. Go to "after activation" below.

= After activation =
1. Go to the settings page and set up the plugin for your site.
1. You're done!
